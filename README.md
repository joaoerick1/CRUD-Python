# CRUD Python com MySQL

Um projeto de backend em Python projetado para realizar operações CRUD (Create, Read, Update, Delete) em um banco de dados MySQL local. O sistema foca na integração segura entre a aplicação e o banco de dados.

## Funcionalidades

* **Create:** Inserção de novos produtos no banco de dados.
* **Read:** Leitura e listagem das vendas.
* **Update:** Atualização de valores de produtos existentes.
* **Delete:** Remoção de registros do banco de dados.
* **Segurança:** Proteção de credenciais do banco usando arquivos `.env`.

## Pré-requisitos

Para rodar este projeto, você precisará de:
* Python 3.x
* MySQL Server
* `mysql-connector-python`
* `python-dotenv`
